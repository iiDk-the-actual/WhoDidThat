# WhoDidThat
A mod for Gorilla Tag that logs Photon-related things.

## How to Use
To use this mod, you need to have the BepInEx console enabled.
All events and RPCs will be sent to your debug console, featuring who sent the command and what arguements they sent it with.
This can be useful for debugging how mods or parts of the game function.

---

# Installation

- Download the release from [here](https://github.com/iiDk-the-actual/WhoDidThat/releases/latest)
- Drag the file `WhoDidThat.dll` to your plugins folder
- Start Gorilla Tag if needed
